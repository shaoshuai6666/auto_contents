# auto_contents
Auto_Contents是一个Python脚本，它基于学位论文的中文目录自动生成**英文目录**。该脚本还包括自动生成**英文摘要**和**英文图表标题**的功能。

## 用法：
1.先安装"pip install python-docx"

2.要使用Auto_Contents，只需运行main.py即可，需要在里面输入文件地址。
